Estilos embebidos en las etiquetas de HTML

Usar estilos CSS embebidos en las etiquetas HTML es el peor y menos utilizados de los metodos para agregar las reglas de estilo. No es nada recomendable por la dificultad que implica realizar un cambio a posterior.

Esta forma de incluir CSS directamente en los elementos HTML solamente se utiliza en determinadas situaciones en las que se deben incluir un stilo muy especifico para un solo elemento en concreto.

Para insertar un estilo CSS en un elemento se ultiliza la propiedad Style de HTML, que en ningun momento se debe confundir con la etiqueta <style< que se introduce en la cabecera del documento.
