Datos Personales

Fortalezas

He dominado el uso de HTML y CSS para crear estructuras web organizadas y atractivas, mejorando notablemente en el diseño de interfaces con un enfoque en usabilidad. Además, el manejo de propiedades CSS como position: "fixed" me permite controlar la disposición de elementos en la página de manera más precisa, lo que contribuye a crear diseños limpios y funcionales.

Oportunidades

Lo aprendido en HTML y CSS me será útil para futuros proyectos, especialmente en el diseño de interfaces y la creación de sitios web eficientes. Estos conocimientos también son aplicables en otros ámbitos profesionales, como la creación de aplicaciones o productos digitales, permitiéndome implementar diseños visualmente atractivos y usables en diversos contextos.

Debilidades

Aunque tengo una buena base en HTML y CSS, aún necesito mejorar en el uso avanzado de CSS Grid y Flexbox para crear diseños más complejos y adaptables. También me gustaría profundizar en la optimización de rendimiento web, como la gestión de la carga de recursos, para mejorar la eficiencia y velocidad de las páginas.

Amenazas

He encontrado obstáculos técnicos, como la falta de experiencia con JavaScript, lo que limita la interactividad en mis proyectos web. Además, el desafío personal de gestionar mi tiempo entre estudios y aprendizaje constante en el desarrollo web ha impactado mi progreso, haciendo más difícil avanzar de manera fluida en todos los aspectos del desarrollo.

Un poco sobre la materia

Materia

Lo que les ha gustado

Lo que más disfruté del curso fue aprender a aplicar HTML y CSS de manera práctica para crear sitios web funcionales y estéticamente agradables. Me resultó particularmente interesante ver cómo pequeñas modificaciones en el código podían tener un impacto significativo en el diseño y la usabilidad de una página. La capacidad de visualizar los cambios de inmediato fue muy motivadora y me permitió explorar con más creatividad.

Lo que no les ha gustado

Lo más desafiante fue la curva de aprendizaje con algunas propiedades avanzadas de CSS, como Grid y Flexbox, que a veces resultaron frustrantes debido a su complejidad. También fue un reto dedicar suficiente tiempo para practicar y dominar estos conceptos en medio de otras responsabilidades académicas, lo cual hizo que el progreso fuera más lento de lo que esperaba.

Retos asumidos

Un desafío importante fue crear una página web con un diseño fijo, donde tuve dificultades con la disposición de los elementos utilizando position: fixed. Logré resolverlo experimentando con diferentes combinaciones de propiedades y ajustando la estructura del código, lo que me permitió superar el obstáculo y mejorar mi comprensión de la disposición de elementos.

Sugerencias para mejorar

Una sugerencia para mejorar el curso sería incluir más ejercicios prácticos enfocados en problemas reales del desarrollo web. También sería útil disponer de más ejemplos en los que se combinen diferentes técnicas avanzadas de CSS para mejorar el entendimiento de cómo se utilizan en proyectos complejos. Además, sería ideal ofrecer más tiempo para revisar y practicar los temas antes de pasar a los siguientes módulos.

Lo que les ha gustado

Algunos recursos valiosos que encontré fuera del curso incluyen tutoriales en MDN Web Docs, que ofrecen guías muy detalladas de HTML y CSS. También utilicé CSS-Tricks para profundizar en temas específicos como Flexbox y Grid. Además, el canal de YouTube Traversy Media fue útil para entender mejor cómo implementar proyectos prácticos. Los artículos de Smashing Magazine me ayudaron a obtener una visión más avanzada sobre diseño y desarrollo web.

  • MDN Web Docs (Mozilla Developer Network)
  • CSS-Tricks
  • Traversy Media (YouTube)
  • Smashing Magazine

Lo que no les ha gustado

Más allá del curso, me gustaría explorar más sobre JavaScript, ya que su integración con HTML y CSS es fundamental para crear sitios web interactivos. También quiero aprender sobre React para desarrollar interfaces más dinámicas, y mejorar en optimización de rendimiento web para crear páginas más rápidas y eficientes. Otro tema que me interesa es accesibilidad web, para asegurar que mis proyectos sean inclusivos para todo tipo de usuarios.
