Guia 1

Esta guía cubre los conceptos fundamentales de HTML y CSS. Los objetivos son que los estudiantes identifiquen la estructura básica de un documento web, apliquen reglas de estilo utilizando hojas de estilo internas y externas, y usen selectores y propiedades CSS básicas. Se explica cómo estructurar un documento HTML, las maneras de incluir CSS (en línea, internas o externas), y el uso de selectores simples como de etiquetas, clases e ID. Además, introduce propiedades CSS esenciales para modificar texto y apariencia.

Ejercicio 1 (estilos embebidos). Ejercicio 2 (Ejemplo 2). Ejercicio 3 (Paises). Ejercicio 4 (Pelicula Favorita).

Guia 2

Esta guía se enfoca en el uso de selectores CSS avanzados y la comprensión del modelo de caja en CSS. Los estudiantes aprenderán a combinar selectores, utilizar pseudo-clases y pseudo-elementos, y manejar el modelo de caja (margin, border, padding, content). Incluye ejemplos de selectores como descendientes, hijos directos, adyacentes, y selectores de atributos. También se explican pseudo-clases como :hover y :nth-child.

Ejercicio 1 (Pelicula). Ejercicio 2 (Tablero). Ejercicio 3 (Biografia 1). Ejercicio 4 (Biografia 2).

Guia 3

Esta guía se centra en el posicionamiento de elementos usando CSS y en las técnicas de maquetación. Se exploran los tipos de posicionamiento (estático, relativo, absoluto, fijo y flotante), y cómo estas técnicas afectan el flujo de la página. También se aborda cómo controlar la visualización de los elementos (propiedades como display, visibility, overflow, y z-index) y el uso de la técnica de flexbox para crear maquetaciones responsivas. Además, cubre cómo solucionar problemas comunes como el desbordamiento del contenido o elementos que no respetan el flujo del documento.

Ejercicio 1 (Oscares). Ejercicio 2 (Menu Horizontal). Ejercicio 3 (Viajes, planes y más). Ejercicio 4 (Analisis).

Guia 4

Esta guía se enfoca en la creación de formularios y el diseño web responsivo usando HTML5 y CSS. Se enseña a crear formularios efectivos y aplicar principios de diseño responsivo utilizando porcentajes y media queries para adaptar el contenido a diferentes dispositivos. Incluye ejercicios prácticos como la creación de formularios alineados sin tablas y el desarrollo de una página de inicio para un portafolio, y finaliza con una sección de reflexión sobre la experiencia y sugerencias para futuras ediciones del curso.

Ejercicio 1. Ejercicio 2. Ejercicio 3. Ejercicio 4. Ejercicio 5. Ejercicio 6. Ejercicio 7 (complementario Nuevo Index).

Guia 5

El objetivo principal de esta guía es que, al finalizar la práctica, el estudiante adquiera un dominio sólido de los elementos sintácticos fundamentales del lenguaje JavaScript. Además, se espera que el estudiante sea capaz de generar código HTML utilizando secuencias de comandos de JavaScript, lo que permitirá crear páginas web interactivas. Otro aspecto importante es que el estudiante podrá aplicar métodos básicos de interacción con el usuario, lo cual es esencial para desarrollar interfaces dinámicas y responsivas. Finalmente, la guía busca que el estudiante utilice de manera adecuada las estructuras selectivas (como las declaraciones condicionales) y las estructuras repetitivas (como los bucles) dentro del contexto de JavaScript, con el fin de optimizar la funcionalidad de las páginas web.
